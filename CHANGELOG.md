# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-09-10

### Added
- Initial release of CallPanion Elderly App
- Native CallKit integration for iOS and Android
- Firebase Cloud Messaging for push notifications
- WebView integration for web-based calling
- Supabase backend integration with Edge Functions
- Scheduled calls functionality
- Pairing mechanism between elderly users and relatives
- Simplified UI designed for elderly users

### Features
- Incoming call notifications with native UI
- Call history tracking
- Device pairing with relatives
- Scheduled call reminders
- Web-based calling interface
- Cross-platform support (iOS and Android)
